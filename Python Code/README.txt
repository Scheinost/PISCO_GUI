The following files must be in the same directory to run PISCO:

PISCO_GUI.py - Main interface file
functions.py - Supporting functions and utilities
icono.png - Window icon image
LogoCkelar.ico - Taskbar icon file


Running Instructions

Ensure all four files are in the same directory 
Open a Python terminal or command prompt
Navigate to the directory containing the files


Dependencies
Make sure you have the following Python packages installed:

tkinter
ttkthemes
PIL (Pillow)
cv2 (OpenCV)
matplotlib
numpy

Optional Hardware
For full functionality:

Raspberry Pi 4
Raspberry HQ Camera
Arduino microcontroller
3D printed components for microscope mounting
